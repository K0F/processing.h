# processing.h

Ideally a single file C library that converts Processing calls into Raylib calls. Libraries are very similar.

# pde2c

A little utility that converts processing pde files into C source code and executable. Using Raylib. 

This is very early proof of concept. Work in progress.

## Prerequesties

 - raylib: https://github.com/raysan5/raylib

## Build

```
	make
	./example
```
